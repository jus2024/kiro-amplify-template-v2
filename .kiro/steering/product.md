# プロダクト方針

このリポジトリは Amplify Gen 2 を使った業務 Web アプリケーション用の再利用可能なスターターテンプレートです。

前提:
- 主な成果物は業務 Web アプリケーション
- エージェント機能はオプション拡張であり、プロジェクトの中心ではない

優先事項:
- 素早いプロジェクト立ち上げ
- 安全なデフォルト構成
- 再現可能な開発ワークフロー
- アプリ本体とオプションのエージェント機能の明確な分離

コード追加時の注意:
- テンプレートは汎用的に保つ
- プロジェクト固有のビジネスロジックをテンプレートに埋め込まない
- 拡張しやすい構造を優先する
